# 문제풀이

- 프로그래머스의 문제를 풀 때 작성한 코드들의 저장소입니다.

- 구성
    - main
        - 문제에 대한 구현 코드가 있습니다.
        - ~~패키지 구성은 코딩테스트 연습> 모든 문제>난이도 별로 구성되어 있습니다.~~
        - 기존 패키지 구조를 변경 중입니다. 테스트 코드는 유지한 채 중복코드를 제거하고 각 테스트 코드에서 호출 시 의미를 파악하기 쉽게하는것이 목적입니다.
    - test
        - 구현 코드에 대한 검증 코드가 있습니다.
        - 패키지 구성은 코딩테스트 연습> 모든 문제>난이도 별로 구성되어 있습니다.
